slant
=====

Minimal 360 Video Player

## install

**npm:**

```js
$ npm install slant
```

**component:**

```js
$ component install littlstar/slant
```

## usage

```js
var src = '/test.webm';
var el = document.querySelector('#video');
var player = Slant(el, {
  src: src,
  height: 400,
  width: 800,
  fov: 28
});

```
